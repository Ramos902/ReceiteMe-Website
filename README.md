# ReceiteMe Website
Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) no curso técnico em Informática. O site tem como objetivo compartilhar receitas personalizadas, 
com filtros específicos para atender pessoas com Doenças Crônicas Não Transmissíveis (DCNTs), facilitando a busca por opções alimentares saudáveis  
e adequadas a diferentes necessidades nutricionais.
